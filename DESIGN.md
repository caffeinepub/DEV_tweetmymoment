# Design Brief

## Tone & Differentiation
Celebratory, inclusive, empowering. Moments worth sharing. Warm, joyful, personal.

## Palette (OKLCH)
| Role | Light | Dark |
|------|-------|------|
| Primary | 0.65 0.16 59 | 0.72 0.18 59 |
| Secondary | 0.42 0.13 149 | 0.65 0.14 149 |
| Background | 0.99 0.01 142 | 0.15 0.01 142 |
| Foreground | 0.18 0.02 142 | 0.95 0.02 142 |

## Typography
Display: Fraunces (premium serif) • Body: Plus Jakarta Sans • Mono: Geist Mono

## Elevation & Depth
Minimal shadows. Air via spacing. Cards elevated on hover (4px). No gradients.

## Structural Zones
| Zone | Treatment |
|------|----------|
| Header | Branding, settings link |
| Event Grid | 2–3 cols, celebration cards (32px radius) |
| Settings | Admin Client ID page |
| Tweet Modal | Full-screen overlay |
| Footer | Legal, links |

## Spacing & Rhythm
Base 8px. Dense mobile, breathing desktop. Cards 16px padding, 24px gap.

## Component Patterns
- Event Card: 32px radius, hover lift, clear icon/label
- Tweet Preview: Large text, styled quote, action buttons
- Input Field: Soft border, 8px radius, gentle focus

## Motion
All transitions 250ms ease-out. Hover lifts 2px. No bouncing.

## Constraints
Mobile-first. Light primary. Dark mode supported. Functional, no decoration.
